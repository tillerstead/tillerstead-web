# Governance Policy

**Tillerstead LLC**
Effective: 2026-03-07

---

## Authority

This repository is owned and governed by **Tillerstead LLC**, a manager-managed New Jersey limited liability company.

All decisions regarding this repository — including but not limited to architecture, licensing, access control, deployment, and intellectual property — are made exclusively by the designated Manager(s) of the LLC.

## Decision-Making

1. **Manager Authority**: The Manager has sole authority over code merges, releases, deployments, access grants, and architectural decisions.
2. **No Implied Voting**: Nothing in this repository or its documentation implies member voting control, community governance, or democratic decision-making over the codebase.
3. **Contributor Submissions**: All contributions (pull requests, issues, suggestions) are advisory. The Manager retains final approval authority.

## Prohibited Actions

The following actions are **expressly prohibited** without written Manager authorization:

- Deletion of repository history, branches, or tags
- Force-pushing to protected branches
- Modification of CI/CD pipeline secrets or deployment targets
- Granting or revoking repository access
- Changing license terms or ownership declarations
- Silent removal of files, features, or documentation without audit trail

## Audit Trail

All significant repository actions must be traceable. This includes:

- Merge commits (no squash-and-delete without record)
- Deployment logs retained for 12 months minimum
- Access changes logged via GitHub audit log

## Amendments

This governance policy may only be amended by the Manager of Tillerstead LLC. Changes take effect upon commit to the default branch.
